**Автор:** Алсу Гильманова

**Дата создания:** Май 2026

## Описание программы

Movie Library — это GUI-приложение для хранения информации о фильмах. Позволяет добавлять фильмы с названием, жанром, годом выпуска и рейтингом, просматривать в таблице, фильтровать по жанру и году, сохранять в JSON.

## Возможности

- Добавление фильмов с названием, жанром, годом и рейтингом
- Просмотр всех фильмов в таблице
- Удаление выбранного фильма
- Фильтрация по жанру
- Фильтрация по году выпуска
- Сохранение данных в JSON-файл
- Загрузка данных из JSON-файла
- Проверка ввода (год — число, рейтинг от 0 до 10)

## Как запустить

python main.py

**Примечание:** Приложение использует стандартный модуль `tkinter`, дополнительные библиотеки не требуются.

## Как пользоваться

1. Заполните поля: название, жанр, год выпуска, рейтинг (0-10)
2. Нажмите "Добавить фильм"
3. Для фильтрации выберите жанр или введите год и нажмите "Применить фильтр"
4. Для удаления выберите фильм и нажмите "Удалить фильм"
5. Для сохранения нажмите "Сохранить в JSON"
